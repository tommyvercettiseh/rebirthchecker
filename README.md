# Rebirth Checker

Windows-widget voor Call of Duty Resurgence maprotaties. Stel de huidige map en resterende tijd in; daarna loopt de rotatie automatisch door. Rebirth Island wordt groen gemarkeerd en kan optioneel een mobiele ntfy-push sturen.

## Starten

Dubbelklik op `Start Rebirth Checker.bat`.

De launcher maakt automatisch een virtuele omgeving, installeert dependencies en start de widget.

## Functies

- Instelbare huidige map en resterende tijd
- Automatische rotatie met standaard 10 minuten per map
- Maps toevoegen, verwijderen en herschikken
- Rebirth Island groen gemarkeerd
- Always-on-top en compacte modus
- System tray met tonen, pauzeren en afsluiten
- Optionele mobiele push via ntfy
- Automatisch opgeslagen instellingen
- Windows EXE-build via GitHub Actions of `Build EXE.bat`

## Mobiele push instellen

1. Installeer de ntfy-app op je telefoon.
2. Abonneer je op een zelfgekozen, moeilijk te raden topic.
3. Vul hetzelfde topic in bij de instellingen van de widget.
4. Gebruik de testknop.
